# 구글 플레이 콘솔 설정 가이드

## 🎯 해결된 문제들

### 1. API 수준 문제 ✅
- **문제**: API 수준 34에서 35로 업데이트 필요
- **해결**: `android/app/build.gradle`에서 `targetSdkVersion 35`로 업데이트
- **상태**: ✅ 완료

### 2. 앱 번들 아티팩트 유형 ✅
- **문제**: 앱 번들 업그레이드 필요
- **해결**: `versionCode 6`, `versionName "1.2.0"`으로 업데이트
- **상태**: ✅ 완료

### 3. 개인정보 처리방침 URL ✅
- **문제**: 개인정보 처리방침 URL 등록 필요
- **해결**: `/privacy` 페이지 생성 및 URL 제공
- **URL**: `https://a-pl.vercel.app/privacy`
- **상태**: ✅ 완료

1. 프로젝트에서 새 APK/AAB 빌드:

3. `android/app/build/outputs/bundle/release/app-release.aab` 업로드

## 🔧 개인정보 처리방침 수정 방법

개인정보 처리방침 내용을 수정하려면 `src/data/privacy-policy-content.ts` 파일을 편집하세요:

```typescript
export const PRIVACY_POLICY_CONFIG = {
  // 회사 정보
  companyName: "에이쁠(A-Pl)",
  serviceName: "에이쁠",
  contactEmail: "support@a-pl.com",
  
  // 처리 목적 (배열에 항목 추가/수정 가능)
  purposes: [
    "회원가입 및 서비스 이용자 식별",
    "AI 퀴즈 생성 서비스 제공",
    // ... 더 추가 가능
  ],
  
  // 수집 정보 (배열에 항목 추가/수정 가능)
  collectedInfo: {
    required: ["이메일 주소", "이름", ...],
    automatic: ["서비스 이용 기록", ...]
  },
  
  // 위탁업체 (배열에 업체 추가/수정 가능)
  processors: [
    { company: "Google LLC", service: "로그인 인증 서비스" },
    // ... 더 추가 가능
  ]
};
```

### 수정 후 배포
1. 파일 수정 후 저장
2. Vercel에 자동 배포됨
3. `https://a-pl.vercel.app/privacy`에서 변경사항 확인
